Java Classes and Objects

This repository contains Java programs based on **Classes and Objects** concept of Object-Oriented Programming (OOP).

Topics Covered
- Class Creation
- Object Initialization
- Constructors
- Methods
- Instance Variables
- Encapsulation Basics
- Real-life Class Examples

Programs Included
- Account Class
- Client Class
- Rectangle Class
- Transaction Class
- Main Driver Programs

Technologies Used
- Java
- IntelliJ IDEA
- Git & GitHub

How to Run
1. Clone the repository
2. Open project in IntelliJ IDEA
3. Navigate to `src` folder
4. Run any `Main` class file

Purpose
This repository is created for practicing and understanding
**Object-Oriented Programming concepts in Java**.
